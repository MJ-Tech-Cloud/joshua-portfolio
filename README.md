## Customization

### Colors
The website uses CSS custom properties for easy color customization. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-color: #008080;    /* Teal */
    --secondary-color: #87CEEB;  /* Light Blue */
    --dark-grey: #2C3E50;
    --light-grey: #ECF0F1;
    --white: #FFFFFF;
}
```
