# Deploy to Imbra - Step by Step Guide

## üöÄ Deploying Your Portfolio on Imbra

### Step 1: Sign Up for Imbra
1. Go to [imbra.com](https://imbra.com)
2. Click "Sign Up" or "Get Started"
3. Create your account with email and password
4. Verify your email address

### Step 2: Choose Your Plan
- **Starter Plan** (Recommended for portfolios)
  - Perfect for personal websites
  - Usually includes 1-5 websites
  - Good bandwidth and storage
  - Professional support

### Step 3: Access Your Control Panel
1. Log into your Imbra dashboard
2. Look for "File Manager" or "cPanel"
3. Navigate to your website's root directory (usually `public_html` or `www`)

### Step 4: Upload Your Files
**Option A: Using File Manager**
1. Open the File Manager in your Imbra control panel
2. Navigate to your website's root directory
3. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `joshua.jpg`
   - `README.md`

**Option B: Using FTP (if available)**
1. Download an FTP client like FileZilla
2. Use your Imbra FTP credentials
3. Connect and upload files to the root directory

### Step 5: Set Up Your Domain
**If you have a custom domain:**
1. Go to "Domains" in your Imbra dashboard
2. Add your domain (e.g., `joshua-mwila.com`)
3. Update DNS settings if needed
4. Point your domain to Imbra's servers

**If using Imbra's subdomain:**
- Your site will be available at: `yourusername.imbra.com`

### Step 6: Test Your Website
1. Visit your domain or subdomain
2. Test all sections:
   - Home page loads correctly
   - Portfolio link works
   - Contact form functions
   - Mobile responsiveness
   - Image displays properly

### Issue: Website not loading
**Solution:**

- Verify domain DNS settings
- Contact Imbra support

### Issue: Images not displaying
**Solution:**
- Ensure `joshua.jpg` is uploaded
- Check file permissions
- Verify file path in HTML

### Issue: Contact form not working
**Solution:**

- Verify form action URL
